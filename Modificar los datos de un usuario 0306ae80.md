### Objetivo

Modificar los [datos personales](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Objeto-Datos-personales-c520636c-ba8b-41f1-b95e-7dd599681bbe) del[ pasajero](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-Husped-Cliente-1aca8769-d624-47f7-9373-9682438afab4).

### Contexto

La cuenta está [activada](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Estado-Activada-ed06a42e-de05-4bbf-a191-8c8a2606554a).

### Recursos

[Datos personales](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Objeto-Datos-personales-c520636c-ba8b-41f1-b95e-7dd599681bbe).

### Actores

[Pasajero](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-1aca8769-d624-47f7-9373-9682438afab4)

### Set de Episodios

1. El [pasajero](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-Husped-Cliente-1aca8769-d624-47f7-9373-9682438afab4) inicia sesión.
2. El [pasajero](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-Husped-Cliente-1aca8769-d624-47f7-9373-9682438afab4) selecciona la opción "Modificar [datos personales](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Objeto-Datos-personales-c520636c-ba8b-41f1-b95e-7dd599681bbe)"
3. Se muestran los [datos personales](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Objeto-Datos-personales-c520636c-ba8b-41f1-b95e-7dd599681bbe) que el [pasajero](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-Husped-Cliente-1aca8769-d624-47f7-9373-9682438afab4) puede editar.
4. El [pasajero](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-Husped-Cliente-1aca8769-d624-47f7-9373-9682438afab4) edita los [datos personales](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Objeto-Datos-personales-c520636c-ba8b-41f1-b95e-7dd599681bbe) deseados.
5. El [pasajero](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-Husped-Cliente-1aca8769-d624-47f7-9373-9682438afab4) selecciona la opción "Guardar cambios".
6. Se guardan los cambios.
7. Se reflejan los nuevos [datos](https://app.nuclino.com/Curso-LEL/Agencia-de-Viajes/Sujeto-Pasajero-Husped-Cliente-1aca8769-d624-47f7-9373-9682438afab4) actualizados.

### Casos alternativos

* Usuario y/o contraseña incorrecto/s.
* El pasajero agrega datos incorrectos (como por ejemplo, un DNI alfabético)
